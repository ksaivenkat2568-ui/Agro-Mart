AgroMart - Direct Farmer to Customer E-Commerce Platform

AgroMart is a web-based digital marketplace designed to bridge the gap between agricultural producers (farmers) and end consumers. By eliminating intermediary-driven supply chains, AgroMart ensures better profit margins for farmers and fair, transparent pricing for consumers.

Features:
Role-Based Dashboards: Distinct environments for Farmers and Customers.
Persistent Data Management: Utilizes web local storage to ensure users, product catalogs, and active carts are saved between sessions.
Real-Time Inventory Listing: Farmers can dynamically list new agricultural produce with custom pricing.
Streamlined Cart & Checkout System: Intuitive interface for consumers to browse, define quantities, and process simulated orders.
Enterprise-Grade Minimalist UI: Built with modern CSS variables, responsive grids, and clean interface layouts for high accessibility.

 Technology Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
Data Storage:Browser `localStorage` API
Architecture:Single Page Application (SPA) DOM manipulation

 How to Run
1. Download or clone this repository.
2. Open `index.html` in any modern web browser (Google Chrome, Firefox, Safari).
3. The application runs entirely on the client-side, requiring no external server or database setup for demonstration.

 Usage Flow
1.Register/Login: Create an account as either a "Farmer" or a "Customer".
2.Farmer Flow:Navigate to the Dashboard, add product names and prices.
3.Customer Flow: Navigate to the Marketplace, view available products, add them to the cart, and proceed to checkout.
